🎬 Dubflix

Dubflix is a Netflix-inspired movie web application designed to showcase films in a clean, modern, and responsive interface. Users can browse through a collection of movies and view detailed information about each title.

🚀 Features

- Movie listing interface (Netflix-style layout)
- Individual movie detail pages
- User authentication (Login/Signup)
- Flash messages for user feedback
- Session-based authentication
- Responsive design

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Templating / Rendering: EJS
- Authentication: express-session
- Flash Messages: connect-flash
